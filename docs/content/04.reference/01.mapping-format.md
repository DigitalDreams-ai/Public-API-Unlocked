# Mapping Format Reference

All mapping values must use this format:

`$intakes.<FieldApiName>`

## Valid Examples

- `$intakes.First_Name__c`
- `$intakes.Last_Name__c`
- `$intakes.Email__c`
- `$intakes.CreatedDate`

## Invalid Examples

- `$submission.Email__c`
- `$intake.Email__c`
- `$PublicApi_Submission__c.Email__c`
- `Email__c`

## Configuration Shape

Stored in `Outbound_Payload_Configuration__c`:

```json
{
  "inbound": [
    { "externalKey": "email", "value": "$intakes.Email__c", "sortOrder": 1 }
  ],
  "outbound": [
    { "externalKey": "email", "value": "$intakes.Email__c", "sortOrder": 1 }
  ]
}
```
