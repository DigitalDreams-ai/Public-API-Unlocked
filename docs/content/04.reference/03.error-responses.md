# Error Responses

## Error Format

```json
{
  "errorMessage": "<message>"
}
```

## HTTP Status Codes

### 400 Bad Request

- Invalid JSON
- Invalid payload shape
- Invalid inbound mapping configuration
- Invalid related-record configuration
- Submission DML/validation failure

Common related-record `400` examples:

- unknown target lookup field
- unknown related object
- unknown related match field
- unsupported `onMultipleMatches` value
- required related-record create fields cannot be satisfied
- multiple related records matched when policy is `Error`

### 401 Unauthorized

- Missing/invalid `X-Api-Key`
- Invalid signature when inbound secret validation is enabled

### 503 Service Unavailable

- API key matches multiple active integration config records

### 500 Internal Server Error

- Unexpected unhandled error
