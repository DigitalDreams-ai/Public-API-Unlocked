# Trigger Field Values

Outbound webhook delivery is driven by the trigger field configured on `PublicApi_Integration_Config__c`:

- `Outbound_Trigger_Enabled__c = true` (or blank, treated as enabled)
- `Outbound_Trigger_Field_Name__c` identifies the watched field on the target object
- `Outbound_Trigger_Field_Values__c` lists the values that should send a webhook

Accepted `Outbound_Trigger_Field_Values__c` formats:

- comma-delimited text
- newline-delimited text
- semicolon-delimited text

Example:

- `Outbound_Trigger_Field_Name__c = Status__c`
- `Outbound_Trigger_Field_Values__c = Converted,Referred Out`

No additional trigger-derived fields are stamped on the source record; the field change is used only to determine whether outbound delivery should run.
