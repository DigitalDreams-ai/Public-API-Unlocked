# Status Values

## Outbound-Trigger Statuses

Status update webhooks are sent only when `Status__c` changes to:

- `Referred Out`
- `Converted`

## Conversion Stamping

When status changes to one of the values above, these fields are set in `before update` for tracked submissions:

- `Google_Ads_Conversion_Event__c = <new status>`
- `Google_Ads_Conversion_Time__c = System.now()`
