# Status Values

## Outbound-Trigger Statuses

Status update webhooks are sent only when `Status__c` changes to a value allowed by the linked `PublicApi_Integration_Config__c`:

- `Outbound_Trigger_Enabled__c = true` (or blank, treated as enabled)
- `Outbound_Trigger_Status_Values__c` includes the new status
  - Accepted format: comma/newline/semicolon-delimited text
  - Accepted format: JSON array (example: `["Referred Out","Converted"]`)
- If `Outbound_Trigger_Status_Values__c` is blank or cannot be parsed, the default set is used:
  - `Referred Out`
  - `Converted`

No additional status-derived fields are stamped on `PublicApi_Submission__c`; status change is used only to trigger outbound delivery.
