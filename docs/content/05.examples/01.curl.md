# cURL Examples

## Create Submission

```bash
curl -X POST "https://<site>/services/apexrest/v1/publicapi/submissions" \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: <api_key>" \
  -d '{
    "first_name": "Jane",
    "last_name": "Smith",
    "email": "jane@example.com",
    "phone": "5551234567",
    "gclid": "test-gclid"
  }'
```

## Create Submission With Signature

```bash
curl -X POST "https://<site>/services/apexrest/v1/publicapi/submissions" \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: <api_key>" \
  -H "x-publicapi-signature: <hmac_sha256_hex>" \
  -d '{"email":"signed@example.com"}'
```
