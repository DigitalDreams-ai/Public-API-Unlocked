# Create Submission

Create a new `PublicApi_Submission__c` record.

## Request

```http
POST /services/apexrest/v1/publicapi/submissions
Content-Type: application/json
X-Api-Key: <api_key>
```

## JSON Example

```json
{
  "first_name": "Jane",
  "last_name": "Smith",
  "email": "jane.smith@example.com",
  "phone": "5551234567",
  "description": "Lead from landing page",
  "gclid": "test-gclid"
}
```

## Form-Encoded Example

```text
first_name=Jane&last_name=Smith&email=jane.smith%40example.com&phone=5551234567
```

## Success Response (201)

```json
{
  "recordId": "a1X...",
  "message": "Submission created successfully"
}
```

## Error Response (400)

```json
{
  "errorMessage": "Failed to create submission: <dml message>"
}
```

## Related Record Errors

If related-record configuration is invalid, the API fails before the target record is inserted.

Example:

```json
{
  "errorMessage": "Related record configuration is invalid: Unknown target lookup field on Task: NotARealLookup__c"
}
```
