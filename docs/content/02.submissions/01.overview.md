# Submissions Overview

`PublicApiIntakeApi` exposes a single inbound endpoint:

```http
POST /services/apexrest/v1/publicapi/submissions
```

## Supported Content Types

- `application/json`
- `application/x-www-form-urlencoded`

## Mapping Behavior

Inbound payload keys map to `PublicApi_Submission__c` fields using `Outbound_Payload_Configuration__c` on the matched integration config.

Canonical field reference format:

`$intakes.<FieldApiName>`

Examples:

- `$intakes.First_Name__c`
- `$intakes.Email__c`
- `$intakes.CreatedDate`

## Automatic Fields Set by API

- `PublicApi_Integration_Config__c` (matched config Id)
- `Created_By_PublicApi_API__c = true`
- `Status__c = New` when status not provided by mapping
- `Raw_Payload__c` (truncated to 131072 chars)
