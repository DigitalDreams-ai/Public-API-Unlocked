# Submissions Overview

`PublicApiIntakeApi` exposes a single inbound endpoint:

```http
POST /services/apexrest/v1/publicapi/submissions
```

## Supported Content Types

- `application/json`
- `application/x-www-form-urlencoded`

## Mapping Behavior

Inbound payload keys map to fields on the configured inbound target object using `Outbound_Payload_Configuration__c` on the matched integration config.

Canonical field reference format:

`$<ObjectApiName>.<FieldApiName>`

Examples:

- `$PublicApi_Submission__c.Status__c`
- `$PublicApi_Submission__c.Received_At__c`
- `$PublicApi_Submission__c.CreatedDate`

## Automatic Fields Set by API

- For `PublicApi_Submission__c`, the API sets:
  - `PublicApi_Integration_Config__c` (matched config Id)
  - `Created_By_PublicApi_API__c = true`
  - `Received_At__c = System.now()`
  - `Status__c = New` when status not provided by mapping
  - `Raw_Payload__c` (truncated to 131072 chars)
- On other inbound target objects, only mapped fields are written.
