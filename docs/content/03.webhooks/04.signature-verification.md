# Signature Verification

## Inbound Signature Verification

For inbound submission requests:

- Set `Webhook_Secret__c` on `PublicApi_Integration_Config__c`
- Send `x-publicapi-signature` with HMAC-SHA256 of the raw request body

If secret is configured and signature is invalid/missing, API returns `401`.

## Outbound Signature

For outbound status webhooks:

- Set `Outbound_Auth_Mode__c = HmacSignature`
- Set `Outbound_Webhook_Secret__c`
- Optional custom header name in `Outbound_Header_Name__c` (default: `x-publicapi-signature`)

Signature is generated from outbound raw payload using HMAC-SHA256.
