# Authentication Issues

## 401 Invalid or missing API key

Check:

- `X-Api-Key` header is present
- Key matches one active integration config record
- No extra spaces in key value

## 503 Misconfigured API key

Cause: multiple active `PublicApi_Integration_Config__c` records with same API key.

Fix:

- Deactivate duplicate records or assign unique keys.

## 401 Invalid webhook signature

Check:

- `Webhook_Secret__c` on matched config
- `x-publicapi-signature` header matches raw body HMAC-SHA256
- Signature is computed from exact request bytes (no reformatting)
