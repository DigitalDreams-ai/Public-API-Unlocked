# Webhook Delivery Issues

## Status change did not send webhook

Verify all are true:

- Status changed to `Referred Out` or `Converted`
- Submission has `PublicApi_Intake_Tracking__c` record
- Linked config has `Outbound_Webhook_URL__c`

## Webhook record shows Failed

Check:

- Endpoint availability
- TLS/certificate validity
- Response status and response body in `PublicApi_Webhook_Delivery__c`
- If error mentions Metadata API / Remote Site Setting, verify outbound delivery ran with a valid Salesforce session context and retry the status update.

## Header problems

Review:

- `Outbound_Auth_Mode__c`
- `Outbound_Header_Name__c`
- `Outbound_Header_Value__c`
- `Outbound_Additional_Headers_JSON__c`

Auth/managed headers are reserved and may override duplicates.
