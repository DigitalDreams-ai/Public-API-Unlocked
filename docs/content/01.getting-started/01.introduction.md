# Introduction

Welcome to the Public API documentation.

This API is a guest-access Salesforce REST endpoint used to create `PublicApi_Submission__c` records and send outbound status-change webhooks.

## What You Can Do

- Create submissions through a public endpoint
- Authenticate requests with `X-Api-Key`
- Optionally validate inbound signatures with `x-publicapi-signature`
- Configure inbound/outbound field mappings from the Request Builder
- Send outbound webhooks when status changes to tracked values

## Base URL Pattern

```text
{site_base_url}/services/apexrest/v1/publicapi/submissions
```

Examples:

- `https://<your-site-domain>/services/apexrest/v1/publicapi/submissions`
- `https://<your-site-domain>/api/services/apexrest/v1/publicapi/submissions` (if your site path prefix is `api`)

## API Version

Current version: `v1`
