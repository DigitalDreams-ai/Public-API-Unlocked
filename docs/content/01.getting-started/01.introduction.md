# Introduction

Welcome to the Public API documentation.

This API is a guest-access Salesforce REST endpoint that records inbound requests on `PublicApi_Submission__c`, then uses the matched `PublicApi_Integration_Config__c` record to create the configured target record asynchronously.

The package is object-agnostic. Target object, record type, mappings, related-record behavior, and outbound webhook behavior are all driven by `PublicApi_Integration_Config__c`.

## What You Can Do

- Create records through a public endpoint
- Authenticate requests with `X-Api-Key`
- Optionally validate inbound signatures with `x-publicapi-signature`
- Configure inbound/outbound field mappings from the Request Builder
- Configure related-record resolution, such as matching or creating an Account or other lookup target
- Send outbound webhooks when status changes to tracked values

## Base URL Pattern

```text
{site_base_url}/services/apexrest/v1/publicapi/submissions
```

Examples:

- `https://<your-site-domain>/services/apexrest/v1/publicapi/submissions`
- `https://<your-site-domain>/api/services/apexrest/v1/publicapi/submissions` (if your site path prefix is `api`)

## API Version

Current version: `v1`
